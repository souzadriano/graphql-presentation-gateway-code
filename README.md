# GraphQL Presentation Code

### Read GraphQL Presentation
https://github.com/souzadriano/graphql-presentation

## Requirements

* Node.js v10.15.0+
* Yarn

## Install

```bash

yarn install

```

## Start Servers

```bash

node server1/index.js
node server2/index.js
node server3/index.js

```

## Start Gateway

```bash

node gateway/index.js

```
